# TexasTech_FecalResistome
This repository contains the code for bioinformatic and R analysis of the paper "A microbiome-independent fecal resistome effect of ceftiofur treatment in dairy cows with metritis"
# General Information
1. Project:  USDA-NIFA-AFRI program (grant number: 2019-67015-29865)
2. Samples: Feces
3. Sequencing method: 16S rRNA gene sequencing & Resistome target enrichment shotgun metagenomics
4. Date: 2023 - 2026
5. Author: Gerardo R. Diaz (diazo005@umn.edu)
6. Description: This dry lab bitacora contains the code and processes followed during the bionformatic analysis, data processing, statistical analysis and data visualization of the paper "A microbiome-independent fecal resistome effect of ceftiofur treatment in dairy cows with metritis" (a.k.a TexasTech Ceftiofur paper)
